# Executor Report Template

Use this template for concise factual reports after a narrow implementation pass.

## Summary
Short statement of what was done.

## Files changed
List the files that were modified.

## What changed
Describe the concrete behavior or code changes without expanding scope.

## Validation
State what was verified and what was not verified.

## Risks / notes
Call out fragility, skipped validation, or follow-up caveats clearly.

## Suggested next step
Only include the next narrow step if it is obvious and directly useful.
