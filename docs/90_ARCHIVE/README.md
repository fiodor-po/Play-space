# 90_ARCHIVE

Здесь живут исторические документы.

Раздел уже используется как canonical bucket для:

- dated audits
- historical baseline docs
- closed chapter bundles
- superseded drafts

Правило раздела:

- dated checkpoint docs и старые baseline документы не держать в верхнем слое `docs/`;
- closed chapter bundles хранить в `03_CLOSED_CHAPTERS/<chapter>/`;
- current truth не хранить здесь.
