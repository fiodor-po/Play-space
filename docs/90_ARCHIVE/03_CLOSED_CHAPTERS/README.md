# 03_CLOSED_CHAPTERS

Здесь живут bundle'ы завершённых chapter tracks.

Целевая форма:

- `docs/90_ARCHIVE/03_CLOSED_CHAPTERS/<chapter>/CHAPTER.md`
- дополнительные `AUDITS/` и `NOTES/`, если они были нужны chapter во время работы

Правило:

- closed chapter больше не хранить в `docs/01_CURRENT_STATE/ACTIVE_CHAPTERS/`;
- chapter bundle переносить сюда целиком;
- archived chapter остаётся историческим reference layer и не управляет текущей работой.
